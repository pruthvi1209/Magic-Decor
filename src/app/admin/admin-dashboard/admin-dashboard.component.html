INSTRUCTIONS

Uploading Layouts

In order to upload Layouts, you are required to upload walls in their respective sequence for a perfect room. Also, while taking the picture for the preview, kindly take the perfect angle for the better display at the user's end.

Uploading Object

Since it is a 3D project, it requires 3D models that are not easily available. To upload the Objects, there are three most important files to work as expected.

  • The js File : It must be in the .js format with json data.

  • The Texture : it is for the texture of the Object

  • The Thumbnail : It must be saved with Thumb.jpg ending
